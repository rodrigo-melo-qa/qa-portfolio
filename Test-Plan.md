# Test Plan – Login Functionality

Target Website: https://practicetestautomation.com/practice-test-login/

## Objective
Verify that users can log in successfully using valid credentials.

## Scope
- Login page
- Input validation
- Error messages
- Logout functionality

## Test Approach
Manual testing

## Test Environment
- Browser: Chrome
- Device: Desktop

## Features to Test
- Login with valid credentials
- Login with invalid credentials
- Login with empty fields
- Logout functionality

## Test Scenarios
1. Successful login with valid credentials
2. Login attempt with incorrect password
3. Login attempt with empty fields
4. Login attempt with invalid username
5. Logout after successful login

## Out of Scope
- Performance testing
- Security testing
